The Thinking needed to build today's Information System

About This Book

Before AI the skills needed to build Digital system was to learn the syntax and semantics of programming languages. It took months to learn and implement using it. But today after AI the learning curve has reduced drastically especially with low code no code tools. Using AI tools like Copilot, Claude, etc we can give the requirements to get the code generated. Coding knowledge is still needed but the way to learn is by using different learning strategies and after learning the techniques to use the acquired knowledge needs new process.

Thank you for taking the time to explore this work. I hope the ideas, insights, and stories within these chapters bring value, clarity, and inspiration.
